Selected work
01 LLM evaluation

LatencyBench

A two-phase benchmark for measuring how delayed information changes LLM behavior: premature commitment, waiting, recovery, and decision stability.

  • Designed experiments over frontier models, public datasets, and multiple delay levels.
  • Defined behavioral metrics including PCR, wait rate, recovery gain, and stability.
  • Built the worker, prompt, dataset, scoring, and deployment scaffolding.
Repository
02 Retrieval systems

Retrieval Value Maps

A measurement framework for identifying when RAG retrieval helps, hurts, or wastes compute on public-knowledge question answering tasks.

  • Implemented BM25, dense retrieval, RRF, reranking, and oracle retrieval paths.
  • Prepared large-scale Wikipedia passage corpora and gold evidence extraction.
  • Analyzed no-op retrieval, judge sensitivity, hedging, and failure taxonomy.
Repository
03 Cloud-native AIOps

Swarm RCA for Microservices

A decentralized and centralized root-cause analysis system for cloud-native microservices using anomaly detection, service graphs, gossip, and Personalized PageRank.

  • Built Kubernetes sidecar agents that exchange rank vectors over gRPC.
  • Used AWS CloudWatch Container Insights and X-Ray telemetry for service graph signals.
  • Created a centralized baseline to compare distributed and centralized RCA behavior.
