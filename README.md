# Shabbir Hussain Portfolio

Static GitHub Pages portfolio for AI systems, research, cloud-native AIOps, retrieval, LLM evaluation, and technical writing work.

## Local Preview

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```

## Publish

Create or use the GitHub repository named `hussain-s.github.io`, push these files to the default branch, and enable GitHub Pages if needed. For a user Pages site, GitHub serves the default branch of `<username>.github.io` automatically.
